## 二、后续建议（按投入产出比排序）

### 优先级 1：P&L 验证 ✅ 已完成

结果：Sharpe 2.16 > 1.0，BREAKOUT + ACCUMULATION 信号均有效（57-59% 胜率）。
详见 `memory/20260425_12_pnl_verify.md`。

**决策**：纳入策略池，继续做因子分组嵌入。

### 优先级 2：因子分组嵌入（中等投入，高潜在收益）

Vision 文档第二节第 2 项。当前 47 个因子全部 Linear 投影到 d_model，模型无法区分因子类别。分组嵌入让模型学到"哪类因子在当前市场更重要"。

**7 个因子组**：
```
entropy:     perm_entropy_m/s/l, entropy_slope, entropy_accel, path_irrev_m/l, turnover_entropy_m/l
volatility:  volatility_m/l, vol_compression, bbw, bbw_pctl, vol_ratio_s, vol_shrink, vol_impulse
eigenvalue:  dom_eig_m/l
coherence:   coherence_l1, purity_norm, von_neumann_entropy, coherence_decay_rate
money_flow:  mf_big_net, mf_big_net_ratio, mf_big_cumsum_s/m/l, mf_sm_proportion, mf_flow_imbalance, mf_big_momentum, mf_big_streak
price:       breakout_range
intraday:    intraday_perm_entropy, intraday_path_irrev, intraday_vol_concentration, intraday_range_ratio
```

**实现**：每组独立 `Linear(group_size → d_group)` → concat → `Linear(sum(d_group) → d_model)`

**改动范围**：仅 `FactorAttentionModel.__init__` 和 embedding 层，不动数据管道。

### 优先级 3：多尺度时间窗（大投入，可能最大杠杆）

当前 seq_len=60 只看日线。加入周线信息（5 日聚合）能捕获中期趋势。

**数据**：项目已有 `scripts/gen_weekly_5d.py` 生成周线

**实现**：
- 日线 60 天 + 周线 12 周 → 双通道输入
- 两个 Temporal Transformer 分别处理 → concat summary token
- 或者用 cross-attention 融合

**改动范围**：数据管道（`build_training_data` 加载周线）+ 模型架构，工作量大。

### 优先级 4：PatchTST 价格分支（大投入，研究性质）

把原始 OHLCV 和因子序列分开处理，再 cross-attention 融合。SOTA 但复杂度高。

**建议**：在 1~3 都做完且 Sharpe > 1.0 之后再考虑。

**下一步行动**：P&L 验证完成 → Sharpe 2.16 达标 → 进入优先级 2（因子分组嵌入）。
