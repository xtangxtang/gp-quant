## 三、关键发现

### 1. Sharpe = 1.05 刚刚过线
- 刚好超过 1.0 的门槛，说明有**独立的 alpha**，不仅仅是市值 beta
- 但不是很高（远低于 Phase 1 报告的 P&L Sharpe 2.16），说明之前的 P&L 确实包含了大量市值 beta 收益

### 2. IC 6.5% 对应的是年化 Sharpe 1.05
- 0.0656 的 Pearson IC 在 613 笔交易中转化为 1.05 Sharpe
- 说明 IC→Sharpe 的转换是合理的，没有明显的信息衰减

### 3. 信号分布极度偏态
- 中位数 -0.58%，均值 +1.93%，最大 +195%
- 说明模型是"截尾型"策略：多数交易微亏，少数交易大赚
- 这不是市值 beta（市值 beta 应该是正态分布的）

### 4. 月度稳定性
- 亏损月份：202503 (-3.27%)、202506 (-2.31%)、202510 (-0.71%)、202512 (-2.72%)、202603 (-0.80%)
- 5 个亏损月 / 16 个月 = 31%，胜率 69%
- 最大连续亏损：202509-202512 (4 个月中 3 个月亏损)

### 5. 所有信号都是 breakout
- accumulation 信号在整个样本期间从未出现
- 说明模型可能只在少数情况下判定为 accumulation 状态

## 四、市值中性化消融的启示回顾

| 设置 | IC | 结论 |
|------|-----|------|
| Phase 1 (无中性化) | 0.0656 | 包含市值 beta |
| Phase 1 (+ 市值中性化) | 0.0136 | 去掉市值后 IC 大幅下降 |

**综合结论**：模型的 alpha 中约 79% 来自市值 beta（小盘股暴露），但剩余 21% 是**真实的选股 alpha**，在 613 笔交易中实现了 Sharpe 1.05。

## 五、后续建议

### 已达成：Sharpe > 1.0 → 纳入策略池
- Phase 1 模型有真实 alpha，可以继续投入

### 下一步方向
1. **增加基本面因子**：用 tushare-fina_indicator/ 的 PE/PB/ROE/营收增速 增强模型区分度
2. **行业中性化**：比市值中性化更全面，防止模型在特定行业内过度集中
3. **样本加权**：剔除 ST、次新、微小盘等低质量样本
4. **增加训练年限**：用 2022-2025 训练 (4 年) 增加有效样本量
