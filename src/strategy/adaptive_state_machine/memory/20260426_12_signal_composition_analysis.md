## 六、关键发现

### 1. 被"排除"的股票恰恰是模型最擅长的领域

ST/次新/微小盘这些理论上被认为是"低质量"的样本，实际表现远超大盘股：
- ST: 胜率 53.7%，平均 +8.21%
- 次新: 胜率 60.5%，平均 +4.48%
- <20亿: 胜率 61.5%，平均 +3.87%

而"高质量"股票（>100亿）: 胜率 43.2%，平均 +1.51%

**模型在这些小市值股票上的 alpha 信号是真实的，不是噪声。**

### 2. 剔除低质量样本会损害基线

原计划第 3 步（样本过滤）的方向是**错误的**。这些股票不是噪声来源，
而是模型选股能力的**主要贡献者**。剔除后：
- 保留类平均收益从 +1.93% 降到 +1.12%（下降 42%）
- 保留类胜率从 47.1% 降到 44.9%
- 总收益只剩 49%，样本数减至 517

### 3. 模型偏中小盘风格

- 中位数市值 60.6 亿（远低于 A 股中位数 ~200 亿）
- 前 5 大盈利交易中，3 只是科创板，2 只是 <50 亿
- sh688585 (+195%) 不是次新/微小盘，而是 162 亿市值的科创板股票

### 4. 市值 beta 不体现在"选小盘"，而体现在"小盘中选对的"

之前的市值中性化消融显示 IC 从 0.0656 降到 0.0136，
说明模型确实有小盘股暴露。但现在的分析表明：
- 模型不是简单选小盘股（否则小盘股胜率应该和随机差不多）
- 而是**在小盘股中识别出了更高信噪比的 alpha 信号**
- 小盘股波动大、关注度高、信息不对称，模型能更好地利用这一点

---

## 七、结论

**跳过原计划第 3 步（样本过滤），直接进入第 4 步（增加训练年限到 2022-2025）。**

理由：
1. 当前信号质量已经很好，过滤反而恶化
2. 模型在中小盘的选股能力是真实 alpha 来源
3. 增加训练年限可以在不改变模型的前提下，让参数估计更稳定
