# 双熵共振 + 分岔捕获策略 — 完整总结

## 一、理论基础

策略建立在**复杂系统物理学**之上，核心假设：市场在关键变盘点前会经历"**熵压缩→临界慢化→分岔爆发**"的相变过程，类似水结冰前的过冷态。

| 概念 | 金融含义 |
|------|---------|
| 低排列熵 (PE↓) | 价格序列高度有序 = 主力在控盘 |
| 正熵差 (PE60 - PE20 > 0) | 短期比长期更有序 = 资金在"锁定"价格结构 |
| 路径不可逆 (PI↑) | 状态转移有因果方向 = 资金在一致性推动 |
| 主导特征值 (\|λ\| → 1) | AR过程失去均值回复 = 系统逼近临界分岔点 |
| 换手熵低 (TE↓) | 日内成交金额集中在少数bar里 = 大单隐蔽吸筹 |

**关键洞察**：主力操作会在两个尺度上留下不同的"熵指纹"——建仓使熵降低（有序化），出货使熵升高（无序化），且日内变化领先于日线。

## 三、Stage 1 — 日线熵计算 (daily_entropy.py)

**输入**: 日线 OHLCV，需60天以上历史

| 指标 | 计算方法 | 作用 |
|------|---------|------|
| PE20 | 20日窗口内收益率序列的 Bandt-Pompe 排列熵 (3阶, 归一化到[0,1]) | 短期有序度；< 0.65 极度有序，> 0.85 混乱 |
| PE60 | 60日窗口排列熵 | 长期参考 |
| 熵差 (Gap) | PE60 - PE20 | 正值 = 短期被"锁定"，资金在组织价格结构 |
| 百分位 (EP) | PE20 在过去120日中的排名 | < 0.40 = 历史极端低位（压缩瓶颈） |
| 路径不可逆 (PI) | 前向/反向状态转移矩阵的 KL 散度 (Seifert) | > 0.02 = 有因果方向；< 0.005 = 随机游走 |
| 主导特征值 (DE) | Yule-Walker AR(2) 伴随矩阵最大特征值 | \|λ\| → 1.0 = 临界慢化（系统失去阻尼） |
| 方差抬升 (VL) | Var(10d) / Var(20d) - 1 | 正值 = 波动率放大（临界前兆） |
| 压缩态 | PE20 < 0.85 AND EP < 0.40 | 连续满足天数越多越强 |

## 四、Stage 2 — 日内熵分析 (intraday_entropy.py)

**输入**: 单日 1 分钟 K 线（~240 bars）

| 指标 | 含义 | 阈值 |
|------|------|------|
| 排列熵 | 全天分钟收益率的有序度 | < 0.65 = 有序 |
| 换手熵 (TE) | 成交量分布的 Shannon 熵 | < 0.88 = 集中（大单吸筹） |
| AM/PM偏移 | PE(下午) - PE(上午) | 负值 = 尾盘更有序 |
| 路径不可逆 | 全天方向性 KL 散度 | > 0.01 = 有因果方向 |
| 活跃度 | 有成交分钟占比 | > 0.70 = 不是僵尸股 |
| Herfindahl | Σ(vol_i/total)² | > 0.008 = 主力集中操作 |
| 振幅比 | (High - Low) / Close | < 0.06 = 窄幅 = 隐蔽吸筹 |

## 五、Stage 3 — 买入融合 (fusion_signal.py)

**核心逻辑**: 日线低熵（压缩态）+ 日内成交突然集中 = **隐蔽建仓**

### 硬性门槛（一票否决）
- 日线百分位 > 0.40 → skip
- 日线 PE20 > 0.85 → skip
- 活跃分钟占比 < 0.70 → skip
- 振幅 > 6% → skip

### 四维评分

| # | 子得分 | 权重 | 衡量什么 | 高分条件 |
|---|--------|------|----------|----------|
| 1 | **日线压缩态** | 30% | 价格是否被"压住" | EP ≤ 40%，PE < 0.85，Gap > 0，连续压缩 > 10天 |
| 2 | **日内集中化** | 30% | 微观成交是否异常集中 | TE < 0.88，TE相比近10日均值↓ > 0.02，Herfindahl > 0.008 |
| 3 | **方向性信号** | 20% | 资金是否有因果方向 | 日内 PI > 0.01，\|λ\| → 0.8，尾盘比早盘更有序 |
| 4 | **量能形态** | 20% | 量能是否配合隐蔽吸筹 | 活跃 > 70%，振幅 < 6%，方差有抬升 |

| 总分 | 信号 |
|------|------|
| ≥ 0.55 | **buy** — 双熵共振确认 |
| ≥ 0.40 | **watch** — 结构有利但未完全共振 |
| < 0.40 | **skip** |

> **物理含义**: PE20 历史低位 → 价格被"锁死"在窄幅（有序态）；日内 TE 突降 → 分散交易突然集中（某人在定向操作）；PI 上升 → 资金流有因果链；振幅小 → 不引起注意。→ 弹簧被压缩到极限，等待释放。

## 六、Stage 4 — 卖出信号 (sell_signal.py)

**核心逻辑**: 买入的镜像——捕捉"有序化瓦解"和"暗中派发"

### 四类卖出场景

#### 1. 熵扩散 (权重 30%) — 结构瓦解
| 指标 | 高分条件 | 含义 |
|------|----------|------|
| PE20 变化速度 | 近5日 > +0.02 | 排列熵快速膨胀 |
| 百分位穿越 | 加速升至 > 70% | 脱离压缩区 |
| 多尺度熵差 | 变负或收窄 | 短期不再比长期更有序 |

#### 2. 暗中派发 (权重 35%) — 日内领先日线（**最有价值**）
| 指标 | 高分条件 | 含义 |
|------|----------|------|
| TE 上升 | 比近期均值高 > 0.02 | 集中→分散，分批出货 |
| AM/PM 反转 | shift > 0 | 上午维持假象，下午出货 |
| PI 下降 | 比近期低 > 0.005 | 资金失去方向性 |
| 集中度↓ | Herfindahl 降低 | 不再有主力痕迹 |

#### 3. 熵衰竭 (权重 20%) — 趋势终结
| 指标 | 高分条件 | 含义 |
|------|----------|------|
| 日线 PE20 | > 0.92 | 完全无序化 |
| 日内 PE | > 0.96 | 分钟级彻底混乱 |
| PI 衰退 | 近5日速度为负 | 方向消失 |
| \|λ\| | → 0 | 不再处于临界态 |

#### 4. 量能异常 (权重 15%) — 辅助确认
- 日内振幅 > 5%、集中度下降、日内熵趋势上升

| 总分 | 信号 |
|------|------|
| ≥ 0.50 | **sell** |
| ≥ 0.35 | **warning**（连续3天 → 强制卖出） |
| < 0.35 | **hold** |

## 七、Stage 5 — 分岔检测 (bifurcation.py)

**目的**: 捕获"熵压缩"模型遗漏的**爆发式突破**（分岔路径），与主路径并行运行。

### Layer 1: 硬预筛门槛（淘汰 99%+ 股票）

```
|λ| ≥ 0.55         系统接近不稳定
加速度 ≤ -0.05     初涨后回调蓄势
流动性 ≥ 300万      机构可进场
20日动量 ≥ -0.08    非自由落体
```

### Layer 2: 分岔评分

| 分量 | 权重 | 衡量什么 |
|------|------|---------|
| 特征值接近度 | 40% | \|λ\| 到 1.0 的距离 + 加速趋势 |
| 熵差异 | 25% | 多尺度 Gap + 百分位低 |
| 方向性 | 20% | PI + 5日加速 |
| 动量质量 | 15% | 正动量 + 负加速（回调蓄势） |

分 ≥ 0.45 → **BUY**，分 ∈ [0.30, 0.45) → **WATCH**

### 趋势护盘 (TrendHoldEvaluator)

分岔买入后，趋势存活期间**压制卖出信号**:
- 卖出阈值 +0.25（从0.50升至0.75才触发卖出）
- 追踪止盈: 峰值回撤 > 20% 且盈利 > 10% → 止盈
- 趋势存活条件: EP < 0.92 AND (\|λ\| > 0.35 OR PI > 0.015 OR 盈利 > 20%)
- 最大持仓延长: 60天

## 八、核心算法

### 排列熵 (Bandt & Pompe 2002)
提取滑动窗口内所有 3 阶序模式 (3! = 6 种)，计算 Shannon 熵后归一化到 [0,1]。
- 不变于单调变换，对噪声鲁棒，捕获因果结构
- 0 = 完美有序，1 = 完全随机

### 路径不可逆 (Seifert 2025)
将收益率离散化为 {-1, 0, +1} 三态（阈值 ±0.5σ），构建 3×3 前向/反向转移矩阵，计算 KL 散度。
- 高 PI = 一致性方向结构（资金有组织）
- 低 PI = 来回震荡（随机游走）

### 主导特征值 (Yule-Walker AR(2))
对中心化收益率拟合 AR(2)，提取伴随矩阵最大特征值。
- |λ| → 1.0 时，AR 过程趋于非平稳 = 市场逼近分岔阈值
- |λ| << 1 时，强均值回复 = 稳定态

### Herfindahl 集中度
$H = \sum_{i=1}^{n} \left(\frac{v_i}{V}\right)^2$，其中 $v_i$ 为第 $i$ 分钟成交量，$V$ 为全天总量。
- H > 0.008 = 成交高度集中（主力操作痕迹）

## 九、组合管理 (v3 回测引擎)

| 机制 | 参数 | 作用 |
|------|------|------|
| 仓位上限 | 15只 | 分散风险 |
| 建仓速度 | 每日最多买2只 | 避免崩盘初期填满仓位 |
| 自适应换仓 | PnL阈值 = max(5%, 分差×2) | 分差大时可替换微利持仓，让强信号进场 |
| 硬止损 | -8% | 控制单票风险 |
| 追踪止盈 | 峰值回撤 > 20% + 盈利 > 10% | 让利润奔跑的同时锁定收益 |
| 趋势护盘 | 卖出阈值+0.25 | 分岔持仓不易被抖下车 |

### 自适应换仓逻辑
仓满时，新候选按分差大小决定可替换范围:
- 分差 ≥ 0.05 且持仓 PnL < 10% → 可换
- 分差 ≥ 0.10 且持仓 PnL < 20% → 可换
- 分差 ≥ 0.15 且持仓 PnL < 30% → 可换
- 选择 entry_score 最低的持仓替换（最弱的先走）

| 维度 | 买入（压缩/建仓） | 卖出（扩散/派发） |
|------|-------------------|-------------------|
| 日线排列熵 | 低位（< 0.85） | 高位（> 0.92）或快速上升 |

| 路径不可逆性 | 上升（有方向） | 下降（丧失方向） |
| AM/PM 偏移 | 尾盘更有序（shift < 0） | 尾盘更混乱（shift > 0） |
| 集中度 | 上升（主力操作） | 下降（散户化） |
| 振幅 | 小（悄然吸筹） | 大（需要出货空间） |
