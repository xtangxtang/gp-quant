# Bull Hunter V11-A — 行业龙头独立通道回测复盘 (2026-04-22)

> 背景: V10 已经证明行业因子能显著提升模型 AUC，但仍错过 CPO / 存储 / 固态设备等主线龙头。为解决“行业趋势跟随 + 龙头确认”的痛点，V11-A 新增 Agent 9 行业龙头独立通道，与 Agent 3 模型通道并行出信号。本次文档记录 V11-A 的完整回测结果、问题诊断和下一步建议。

## 1. 本次实验做了什么

### 1.1 新增 Agent 9 行业龙头独立通道

两阶段筛选:

1. **行业热度筛选**
   - `industry_rs_20d >= 4%`
   - `industry_breadth_5d >= 55%`
   - `industry_mom_20d >= 5%`
   - `industry_vol_surge >= 1.1`
   - 行业样本数 `N >= 4`
   - 取 Top 6 热门行业

2. **行业内龙头筛选**
   - `momentum_20d >= 10%`
   - `momentum_5d >= 0`
   - `close_vs_high_60d` 在 `[0.85, 1.05]`
   - `vol_ratio_s >= 1.2`
   - 相对行业强度 `rs_vs_industry >= 5%`
   - `path_irrev_m >= 0`

### 1.2 Pipeline 集成

- Agent 9 在 Agent 3 之后运行，输出独立候选池
- `merge_with_model_predictions()` 合并模型通道和 IL 通道，去重后最多保留 8 只
- `prob_200` 过滤改成**仅作用于模型通道**
- `no_buy_today` / 弱模型 / 大盘过滤改成**仅清空模型通道**，行业龙头通道豁免
- Agent 5 中 `prob_200` 强制门槛也改成**仅模型通道生效**

## 3. 核心结果

### 3.1 总体表现

| 指标 | V11-A |
|------|-------|
| 交易笔数 | 149 |
| 胜率 | 35.6% |
| 平均盈亏 | +1.3% |
| 总盈亏 | -191,033 |
| 平均持有 | 16.2 天 |
| 最佳单笔 | +90.8% |
| 最差单笔 | -29.9% |

**结论**: V11-A 整体失败。信号数量大幅增加，但收益质量显著恶化。

### 3.2 通道拆解（稳定期 `>= 2025-02-25`）

> 为排除 cold-start 初期模型不可用的噪音，单独统计模型稳定后的阶段。

| 通道 | 交易笔数 | 胜率 | 平均盈亏 | 总盈亏 | 平均持有 |
|------|----------|------|----------|--------|----------|
| 模型通道 (`prob_200 > 0.05`) | 42 | 59.5% | +13.1% | +401,075 | 28.1 天 |
| IL 通道 (`prob_200 <= 0.05`) | 89 | 24.7% | -3.4% | -462,218 | 10.4 天 |

**结论**: 模型通道仍然有效，V11-A 的净亏损几乎完全来自 IL 通道。

### 3.3 与 V10 的重叠期对比（`2025-04 ~ 2025-12`）

| 版本 | 交易笔数 | 胜率 | 平均盈亏 | 总盈亏 | 平均持有 |
|------|----------|------|----------|--------|----------|
| V10 纯模型 | 15 | 93.3% | +30.5% | +587,423 | 84.6 天 |
| V11 全部 | 81 | 35.8% | +3.2% | +98,836 | - |
| V11 模型通道 | 34 | 55.9% | +11.4% | +230,614 | 27 天 |
| V11 IL 通道 | 47 | 21.3% | -2.7% | -131,778 | - |

**结论**:

- IL 通道不仅自身亏钱，还明显**稀释了模型通道质量**
- V10 的持有风格是“少而准、持有更久”，V11-A 变成了“高频试错、仓位拥挤”

## 4. 关键证据

### 4.1 Agent 9 出信号过于频繁

- Agent 9 总运行次数: `324`
- 有候选的次数: `236`
- 无热门行业: `88`

也就是 Agent 9 在 **73%** 的交易日都能找到“热门行业 + 龙头候选”。

这与“独立龙头信号”应有的稀缺性不匹配，说明 Phase 1 和 Phase 2 门槛都偏松。

### 4.2 IL 候选经常单独主导买入池

- 有龙头候选的日期: `236` 天
- 有模型候选的日期: `166` 天
- **仅龙头通道有候选**: `70` 天

说明有大量交易日实际上是 IL 通道单独在驱动组合暴露，而不是给模型通道做补充。

### 4.3 IL 通道的亏损结构非常差

IL 通道卖出原因统计:

- `stop_loss`: 49 笔，占 55%，平均 `-11.7%`
- `exit_signal`: 26 笔，占 29%，平均 `-4.8%`
- `trailing_stop`: 13 笔，占 15%，平均 `+29.2%`

拆开看:

- IL 亏损交易: `67 / 89 = 75%`
- IL 亏损合计: `-938,616`
- IL 盈利合计: `+476,398`
- IL 净结果: `-462,218`

这说明 IL 通道不是“没有 alpha”，而是**正样本太少、噪音太多、坏交易远多于好交易**。

### 4.4 亏损集中行业

IL 通道亏损最重的行业:

| 行业 | 交易数 | 平均盈亏 | 总盈亏 |
|------|--------|----------|--------|
| 软件服务 | 6 | -7.0% | -66,368 |
| 渔业 | 4 | -19.2% | -61,126 |
| 黄金 | 5 | -7.9% | -58,171 |
| 铜 | 3 | -8.9% | -52,835 |
| 机械基件 | 5 | -6.0% | -49,533 |
| 农药化肥 | 1 | -24.0% | -47,610 |

这些行业的共同特征是: 题材轮动快、脉冲强、回撤也快。当前 IL 通道缺少足够强的质量过滤和节奏控制。

## 5. 为什么 V11-A 会失败

### 5.1 Phase 1 行业热度门槛太低

当前只要求:

- `industry_rs_20d >= 4%`
- `industry_breadth_5d >= 55%`
- `industry_mom_20d >= 5%`
- `industry_vol_surge >= 1.1`

这会让很多普通轮动板块也被误判成“热门主线”。

### 5.2 Phase 2 龙头确认仍然偏松

当前允许:

- 仅 `momentum_20d >= 10%`
- `close_vs_high_60d >= 0.85`
- `path_irrev_m >= 0`

这更像“行业里相对强的股票”，还不是“主升浪龙头”。

### 5.3 IL 没有自己的质量门控

虽然 IL 通道豁免了模型的 `prob_200` 过滤，但没有建立足够严格的替代质量门槛，导致很多中等质量甚至低质量标的直接进入 Agent 5。

### 5.4 IL 与模型通道抢同一组仓位

从结果上看，V10 的成功来自:

- 交易少
- 胜率高
- 持有时间长
- letting winners run

而 V11-A 引入 IL 后:

- 交易数大增
- 持有期缩短
- 仓位被短周期、低质量 IL 信号占用

这不是“补充通道”，而是“改变了组合风格”。

## 6. 建议

### 6.1 近期结论: 不建议将当前 IL 通道作为默认主通道继续使用

如果当前目标是维持 V10 的稳定盈利能力，建议:

1. 默认关闭 `enable_industry_leader_channel`
2. 保留 Agent 9 代码，但作为实验性分支继续迭代
3. 后续所有 IL 改动必须先做 ablation，再决定是否并回主配置

### 6.2 如果继续探索，建议走 V11-B（强约束 IL）

#### 建议 A: 大幅收紧 Phase 1 行业热度门槛

建议至少改成:

```python
industry_rs_20d >= 0.08      # 甚至 0.10
industry_breadth_5d >= 0.70
industry_mom_20d >= 0.08
industry_vol_surge >= 1.30
top_k_industries <= 2 or 3
```

目标: 让“热门行业”回到稀缺状态，而不是每天都有 5-6 个。

#### 建议 B: 收紧 Phase 2 龙头确认

建议改成:

```python
momentum_20d >= 0.15
momentum_5d >= 0.02
close_vs_high_60d >= 0.92
vol_ratio_s >= 1.50
rs_vs_industry >= 0.08
path_irrev_m >= 0.02
```

目标: 只保留已经显著脱颖而出的行业龙头，而不是行业内普通强势股。

#### 建议 C: 为 IL 通道单独增加质量门控

建议增加至少一层:

- `buy_quality >= 0.50`
- 或 `il_score >= 某阈值`
- 或要求同时满足 `market_state in {UP, NEUTRAL}`
- 或弱模型闸门生效时，只允许 IL 的 Top 1-2 名通过

#### 建议 D: 限制 IL 仓位上限

建议:

- IL 持仓上限 `<= 2`
- 每行业最多 `1` 只
- 模型通道始终保留至少 `4~6` 个仓位

这样 IL 才是补充，而不是反客为主。

#### 建议 E: 针对高波动题材加冷却/黑名单

从这次结果看，以下行业应当优先纳入冷却逻辑:

- 渔业
- 黄金
- 铜
- 农药化肥
- 软件服务

可以考虑:

- 最近 20 笔该行业 IL 交易若累计为负，则暂停 20 个交易日
- 或者将这些行业默认视为“需要更高阈值”的高波动行业

### 6.3 下一轮实验建议

推荐按下面顺序做 3 组最小化实验:

1. **V11-B1: 强约束 IL + 最多 2 个 IL 仓位**
2. **V11-B2: 强约束 IL + 仅在模型弱势时启用**
3. **V11-B3: 纯 IL 通道单独回测**

如果 B1/B2/B3 都不能把 IL 通道拉回正收益，就说明“行业趋势跟随 + 龙头确认”这个思路需要重做目标定义，而不是继续微调阈值。

## 7. 本次结论

V11-A 回答了一个非常关键的问题:

> **“行业趋势跟随 + 龙头确认”值得探索，但当前这版 Agent 9 还不能直接交易。**

更准确地说:

- **方向不是错的**，因为 IL 通道确实跑出了少数很强的大赢家
- **实现是错位的**，因为热度判断过松、质量门槛不足、仓位竞争失控

因此，本轮实验的最佳产出不是“继续放大 Agent 9”，而是:

1. 保留 Agent 9 作为研究分支
2. 回到 V10 作为当前主 baseline
3. 用更严格的行业筛选和仓位约束做下一轮 ablation

一句话总结:

**V11-A 证明了 IL 通道当前版本是负贡献，但也证明了“板块主线 + 龙头确认”里确实藏着 alpha；下一步不是放弃这个方向，而是把它从“高频噪音源”压缩成“低频高质量补充信号”。**
