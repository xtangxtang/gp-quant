# Bull Hunter V9 — 大盘过滤 + 门槛提升 (2026-04-21)

> 基于 V8 回测问题分析 (-22.7% 最大回撤集中在 2025-04)，落地两项改进并通过完整 live_backtest 验证。

## 1. 改进动机 (来自 V8 分析)

## 2. 实施内容

### 2.1 建议①：大盘趋势过滤

**数据来源**：`results/market_trend/market_trend_daily.csv` (554 天，20240102-20260420)
- 状态分布：NEUTRAL=248 / DOWN=226 / UP=46 / STRONG_DOWN=34
- 确认 2025-04-07/08 = STRONG_DOWN（V8 重灾区）

**过滤规则**：
```python
market_no_buy_states        = ("STRONG_DOWN",)       # 暂停新买入
market_tighten_stop_states  = ("DOWN", "STRONG_DOWN") # 收紧止损
market_tightened_stop_pct   = -0.08                   # -8% (默认 -15%)
```

**代码位置**：[pipeline.py](../v3_bull_hunter/pipeline.py) `run_live` 方法，模型加载后、Agent 3 之前：

```python
# 弱模型闸门
if _auc < cfg.min_model_auc:
    no_buy_today = True
    logger.info(f"  ⚠️ 弱模型闸门: AUC={_auc:.3f} < {cfg.min_model_auc:.2f}")

# 大盘过滤
if market_state in cfg.market_no_buy_states:
    no_buy_today = True
    logger.info(f"  🛑 大盘过滤: {market_state} → 暂停新买入")

if market_state in cfg.market_tighten_stop_states:
    portfolio_cfg = replace(portfolio_cfg, stop_loss_pct=cfg.market_tightened_stop_pct)
    logger.info(f"  🛡️ 大盘过滤: {market_state} → 收紧止损 -8%")
```

Agent 5 前清空买入候选：
```python
if no_buy_today and not predictions.empty:
    predictions = predictions.iloc[0:0]
```

## 7. 已知遗留问题

1. **早期仍亏 -60K** (26笔 42% 胜率)：弱模型期 AUC=0.50 时即使 prob_200=0.30 门槛也过宽（早期概率集中在低区间）。
2. **prob_200 判别力弱**：赢家 0.410 vs 输家 0.419（几乎相同），模型信号仍不显著。
3. **4 个行业持续亏损**：元器件/电气设备/机械基件/化纤。
