# Rapid Expansion Exhaustion Exit

## 描述

一句话概括：专门抓快速扩张型上涨在末端出现降速、推力衰减和结构脆弱化的退出时刻。

这套策略不是判断“有没有上涨”，而是判断“强拉升是否已经进入衰竭区”。

核心思想：

- 先确认前面确实经历过一段快速扩张
- 再检测当前是否出现推力回落和节奏降速
- 最后确认这种降速是否已经和结构脆弱性叠加，形成真正的退出信号

主要看四类量：

- `peak_extension_score`：上涨是否已经进入高位扩张末端
- `deceleration_score`：扩张推力、方向连续性和承接是否在降速
- `fragility_score`：高熵、高波动和相位不稳是否开始积累
- `exhaustion_exit_score`：末端衰竭退出的综合强度

## 主要参数

- `symbol_or_name`：股票代码或股票名称，例如 `sh688268`、`688268.SH` 或 `华特气体`
- `start_date`：起始持有日期，格式 `yyyymmdd`
- `scan_date`：评估日期，默认自动推断最新交易日
- `data_dir`：日线 CSV 目录
- `basic_path`：`tushare_stock_basic.csv` 路径，用于名称和代码映射
- `lookback_years`：特征回看年数，默认 5
- `exit_persist_days`：连续多少天满足衰竭条件才判定退出，默认 2

## 输出

策略会输出：

- 当前是否已经进入快速扩张末端的衰竭退出区
- 首次退出确认日期（如果存在）
- 高位扩张分数、降速分数、脆弱度分数和综合退出分数
- 一份完整的日度诊断时间序列 CSV
