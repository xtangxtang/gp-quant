# Rapid Expansion Hold

## 描述

一句话概括：专门针对快速扩张型上涨，判断一只股票在强趋势扩张阶段是否仍然值得继续持有。

这套策略不是低熵压缩后的启动策略，也不是严格的低熵持有策略。它承认快速上涨阶段常常伴随高熵、高波动和较弱的传统“记忆”读数，因此会额外检查扩张推力、方向持续性和市场接受度，避免把所有高熵上涨都误判成失控。

核心思想：

- 高熵不一定等于坏事，关键看是否仍有扩张推力
- 允许快速拉升阶段出现更高的波动和更快的状态切换
- 真正退出要看推力衰减、承接消失和风险累积是否同时发生

主要看四类量：

- `expansion_thrust`：价格、量能和资金流的扩张推力
- `directional_persistence`：上涨路径是否仍然具有方向一致性
- `acceptance_score`：市场是否仍在高位承接这轮扩张
- `instability_risk`：高熵与高波动是否已经开始压过扩张本身

## 主要参数

- `symbol_or_name`：股票代码或股票名称，例如 `sh688268`、`688268.SH` 或 `华特气体`
- `start_date`：起始持有日期，格式 `yyyymmdd`
- `scan_date`：评估日期，默认自动推断最新交易日
- `data_dir`：日线 CSV 目录
- `basic_path`：`tushare_stock_basic.csv` 路径，用于名称和代码映射
- `lookback_years`：特征回看年数，默认 5
- `exit_persist_days`：连续多少天满足退出条件才判定无法继续持有，默认 2

## 输出

策略会输出：

- 当前是否建议继续按“快速扩张持有”逻辑持有
- 首次失效日期（如果存在）
- 扩张推力、方向持续性、承接强度、风险强度
- 一份完整的日度诊断时间序列 CSV
